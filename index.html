SyntaxhighligherLite

一个快速上手的代码高亮组件

访问 GitHub

特性

  • 无需调用任何 API ，引 1 个 js 文件即可高亮。
  • 仅 8k (Gzip 压缩) 。
  • 兼容主流浏览器（包括 IE6）。
  • 异步高亮，不影响页面渲染。
